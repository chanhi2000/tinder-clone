## 실행 커맨드 목록 

#### `npm start`

 - 개발모드에서 실행
 - 확인방법 : [http://localhost:3000](http://localhost:3000) URL호출 


#### `npm test`

 - 테스트 환경에서 실행 

#### `npm run build`

 - 운영 환경용 결과물 생설
